# 环境安装
[https://golang.google.cn/dl/](https://golang.google.cn/dl/)
![image.png](/images/dev/go/go_basic/go_installer.png)
安装会自动将`Go/bin`添加到环境变量。

**测试**

在工作目录中创建`test.go`文件。
```go
package main

import "fmt"

func main() {
   fmt.Println("Hello, World!")
}
```
打开命令窗口，输入`go run test.go`
