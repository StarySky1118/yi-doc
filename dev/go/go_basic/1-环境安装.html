一、环境安装

https://golang.google.cn/dl/ 安装会自动将Go/bin添加到环境变量。

测试

在工作目录中创建test.go文件。

package main

import "fmt"

func main() {
   fmt.Println("Hello, World!")
}

打开命令窗口，输入go run test.go
